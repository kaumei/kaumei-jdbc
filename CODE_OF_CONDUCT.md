# Code of Conduct

This project aims to keep a respectful and technically productive environment.

Please:
* be respectful and considerate
* stay focused on technical topics and project goals
* assume good intent

Unacceptable behaviour includes harassment, insults, aggressive conduct, spam,
and repeated disregard of project guidelines.

This applies to issues, discussions, pull requests, and official project spaces.

The maintainer may remove content or close threads to keep the project workable.

Please do not raise Code of Conduct concerns in public threads.
If you observe a violation, contact the maintainer privately.
