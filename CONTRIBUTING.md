# Contributing to **Kaumei&nbsp;JDBC**

Thanks for your interest in **Kaumei&nbsp;JDBC**.
This project is maintainer-driven and kept intentionally focused.

---

## Where to start
* **GitHub Issues:** concrete, actionable items (confirmed bugs, clear tasks)
* **GitHub Discussions:** questions, ideas, and design proposals

If a discussion turns into a clear task, it can become an issue later.

---

## Pull requests

Please discuss non-trivial changes first.
Keep PRs small, focused, and aligned with the project direction.

---

## Bug reports

Please include:

- **Kaumei&nbsp;JDBC** version
- database and JVM version
- minimal reproduction if possible
- expected vs actual behaviour

---

Final decisions about scope and direction are made by the maintainer.
